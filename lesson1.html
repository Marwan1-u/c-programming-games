Lesson 1: The Massive Inventory System

🧟 Resident Evil Logic: The Attaché Case

Every item in Leon's case is a Variable. In C, we have to tell the computer exactly what "Type" of item it is before we store it.

The Full Inventory Code:


#include <stdio.h>

int main() {
    // 1. INTEGER (Whole Numbers) - Ammo and Health
    int handgun_ammo = 15;
    int green_herbs = 2;
    int pesetas = 5500;

    // 2. FLOAT (Decimal Numbers) - Knife Durability
    float knife_condition = 92.5f;
    float body_armor = 100.0f;

    // 3. CHAR (Single Letters) - Performance Rank
    char mission_rank = 'S';

    // --- GAME LOGIC SIMULATION ---
    printf("--- LEON'S STATUS ---\n");
    printf("Ammo: %d | Rank: %c\n", handgun_ammo, mission_rank);
    printf("Wallet: %d Pesetas\n", pesetas);
    printf("Knife: %.1f%% condition\n", knife_condition);

    // Logic: A Zombie attacks!
    printf("\n[ALERT]: A Ganado grabbed you!\n");
    body_armor = body_armor - 15.5; // Decimal math
    handgun_ammo = handgun_ammo - 3; // Integer math

    printf("Updated Armor: %.1f%%\n", body_armor);
    printf("Remaining Ammo: %d\n", handgun_ammo);

    return 0;
}
