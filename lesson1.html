Lesson 1: Variables (The Inventory System)

In C, a Variable is a container for data. Think of Leon's attaché case in Resident Evil 4...

🧟 Resident Evil Logic

Every time Leon picks up herbs or ammo, the game updates a variable. If health = 100 and a zombie bites you, the code runs health = health - 20.

The Code Example:

int health = 100; // Whole number
float knife_hp = 85.5; // Decimal number
char rank = 'A'; // Single letter
