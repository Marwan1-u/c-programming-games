Level 4: Pointers & Memory

In Elden Ring, your character doesn't just "exist." They occupy a specific spot in your computer's RAM (Memory). To find you, the game uses Pointers.

✨ The Site of Grace Analogy

Imagine your Runes are stored in a chest at a specific Site of Grace.

  • The Variable: The actual Runes inside the chest.
  • The Pointer: The map marker that tells you *where* the Site of Grace is.
In C, a pointer doesn't hold a value like "100"; it holds a Memory Address (the map coordinates).

The Master Code

int runes = 5000; // The actual data
int* ptr_to_runes; // The pointer (Map Marker)

// Store the "address" of runes into the pointer
ptr_to_runes = &runes;

// Accessing data via the pointer (Dereferencing)
printf("You have %d runes at address %p", *ptr_to_runes, ptr_to_runes);
