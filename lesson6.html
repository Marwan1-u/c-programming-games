Lesson 6: The Outlaw System

Mastering Structs and Data Grouping in C

What is a Struct?

In Red Dead Redemption 2, Arthur has health, stamina, and a bounty. In C, we use a Struct to bundle these different types of data (int, float, char) into one single "Package."

The Outlaw Code:

#include <stdio.h>

// The "Package" for our Outlaw
struct Outlaw {
    char name[20];
    int health;
    float money;
    int bounty;
};

int main() {
    // Initializing Arthur
    struct Outlaw arthur = {"Arthur Morgan", 100, 150.50, 0};

    printf("Name: %s\n", arthur.name);
    printf("Bounty: $%d\n", arthur.bounty);

    // Logic: Commit a crime!
    arthur.bounty += 250; 
    printf("\n[!] CRIME COMMITTED! New Bounty: $%d\n", arthur.bounty);

    if(arthur.bounty > 200) {
        printf("Status: WANTED DEAD OR ALIVE\n");
    }

    return 0;
}
