Level 2: The Decision Tree

In Detroit: Become Human, every choice matters. But how does the computer know which ending to show you? It uses Control Flow.

🤖 Mission: Interrogation

You are Connor. You have found a hidden diary. In C code, we check if the player has the item using an if statement. If the condition is true, one thing happens. If not (else), something else happens.

The Code Logic

// 1 = Found Clue, 0 = Missed Clue
int found_diary = 1;

if (found_diary == 1) {
    printf("Connor: I found your diary. You're lying.");
} else {
    printf("Connor: I have no evidence... yet.");
}

Key Takeaway

  • IF: The "Gatekeeper." Only runs code if the condition is met.
  • ELSE: The "Safety Net." Runs code if the condition fails.
  • BRACES { }: These hold the "consequences" of the choice.
